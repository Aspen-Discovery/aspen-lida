Aspen LiDA

Uses expo.io to build https://expo.io/

To setup for development

1) Install Node.js - see: https://nodejs.org/en/download/
2) Navigate to the code directory 
3) Run yarn from the code directory to retrieve dependencies
4) Copy templates in app-config-templates to app-configs and substitute your information
5) Copy the relevant start and updater scripts from script-templates to scripts and update as needed. 

Run the start script in scripts to run LiDA locally. To debug, you may need to first have a developer build within expo. 

Run the updater script in scripts to create and update builds for LiDA using expo. 

Optional
- Install Android Studio - https://developer.android.com/studio
